Amal's Online Continuing Professional Development (CPD) Tracker

Hi I am Amal and welcome to my online CPD tracker page which I use to keep track of my progress as a Software Developer. In my journey to become a well rounded software developer I have tried many different methods of learning software development, i.e. Watching tutorials on Youtube, following online courses on platforms such as Udemy, reading tutorials, searching on Google and by doing software projects that I like. One thing I realized quickly is that if I don't apply what I learn, I tend not to understand the concepts and learning does not stick. I love solving real world problems and when I encounter a problem I tend to break it down into small components and then try to find the optimal method to solve the problem most efficiently as every problem can be solved in so many different ways. I think the passion I have for Software development comes from my natural tendency to understand how the World works which has allowed me to have wide variety of interests ;

  • Electronics Engineering: Which led me to get my degree in Bachelor of Engineering in Electronics Telecommunications and Internet Engineering.
  • CyberSecurity: With the constant threat of Cyber incidents happening in the world and how much it is related to the widespread usage of technology in peoples lives led me to finish a "Graduate Certificate in CyberSecurity and Networking".
  • Computers, Linux, Android phones and Microsoft Windows Technologies: Since at a young age I greatly enjoyed tinkering around with computers which led me to build few desktop computers, diagnose and trouble shoot Windows problems, learn Linux and also have an interest in Android Phones.
  • Financial Markets: Which led me to study my "Diploma in Financial Planning" and to work as a Paraplanner in Financial planning in Melbourne CBD.

My goal is to learn something new everyday and to keep improving myself !

The Odin Project course progress Tracker

After looking at so many different courses available Online and the bootcamps available from leading Australian Universities and other boot camp providers I have finally found everything I need to learn as a Web/Software Developer in Online Open Source Boot Camp: The Odin Project.

Thing I love about "The Odin Project" is that how it takes you on a journey of learning Software Development not only on the concepts by teaching and by doing. Course encourages the student to work as a real world Software developer would, by using the same tools and techniques i.e. Git, Virtual Machines, Linux, Clean Code, problem solving, VS Code.

The Odin Project is a High quality Web Development coding education created by an Open-Source community.

Odin Project: Foundations Path

A hands-on introduction to all of the essential tools you'll need to build real, working websites. learn what web developers actually do in real life and practice using essential tools such as Git, Linux command line,Text editors (VS Code), Ubuntu Linux Virtual Machine Development environment. Also learn the basics of HTML, CSS and JavaScript.
